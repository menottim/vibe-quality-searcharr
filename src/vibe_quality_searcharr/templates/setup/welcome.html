Welcome to Vibe-Quality-Searcharr

Intelligent backlog search automation for Sonarr and Radarr

What is Vibe-Quality-Searcharr?

Vibe-Quality-Searcharr automatically searches your Sonarr and Radarr backlog for unwatched or missing content, intelligently prioritizing items based on your preferences and viewing habits.

Key Features

  • Automated backlog searching with customizable strategies
  • Support for both Sonarr (TV shows) and Radarr (movies)
  • Secure API key encryption and authentication
  • Scheduled and recurring searches
  • Configuration drift detection
  • Comprehensive search history and statistics

Getting Started

This setup wizard will guide you through:

  1. Creating an admin account
  2. Connecting your first Sonarr or Radarr instance
  3. Configuring basic settings
